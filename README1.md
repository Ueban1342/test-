# 💅 Бот для записи к мастеру маникюра

Полнофункциональный Telegram-бот для записи клиентов + веб-админка для мастера.

## 📦 Технологии
- Python 3.10+
- Telegram Bot API через `pyTelegramBotAPI`
- SQLite (локальная база `bot.db`)
- Streamlit (веб-админка)
- APScheduler (напоминания)
